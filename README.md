# PinoySeoul - Link in Bio

A lightweight, self-hosted link-in-bio page for PinoySeoul Media Enterprise.

Built with [LittleLink](https://github.com/sethcottle/littlelink) - a DIY open source LinkTree alternative.

## Deploy

This repo is designed to be hosted on GitHub Pages at `https://pinoyseoul.github.io`.

### Enable GitHub Pages

1. Go to **Settings** > **Pages**
2. Under **Source**, select **Deploy from a branch**
3. Select **main** branch and **/(root)** folder
4. Click **Save**

Your site will be live at `https://pinoyseoul.github.io`.

## Customization

- **Links**: Edit `index.html` to add, remove, or modify buttons
- **Avatar**: Replace `images/avatar.png` and `images/avatar@2x.png` with your own
- **Theme**: Change `class="theme-auto"` to `theme-light` or `theme-dark` in `index.html`
- **Colors**: Edit CSS variables in `css/style.css`
